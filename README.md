# TexWelder

this is a node-based procedrual texture maker made with raylib. 
inspired by the __Blender Material Editor__ and the __Unity Shader Graph System__

the nodes are processed by glsl shaders

this is still a work in progress project and is gonna be improved.

click [here](https://github.com/TahaVr12/TexWelder/blob/84f19c33057740ac4a79cacc84b1021ceb731d9b/BUILDINSTRUCTIONS.md) for build instructions
